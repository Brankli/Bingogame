# ⚡ Any Line Pattern - Quick Guide

## What is it?
A new bingo pattern where you win by completing **ANY single line** - horizontal, vertical, or diagonal.

## 12 Ways to Win! 🎯

### 5 Horizontal Rows:
```
Row 1: [X][X][X][X][X] ← Win!
Row 2: [ ][ ][ ][ ][ ]
Row 3: [ ][ ][F][ ][ ]
Row 4: [ ][ ][ ][ ][ ]
Row 5: [ ][ ][ ][ ][ ]
```

### 5 Vertical Columns:
```
[X][ ][ ][ ][ ]
[X][ ][ ][ ][ ]
[X][F][ ][ ][ ]
[X][ ][ ][ ][ ]
[X][ ][ ][ ][ ]
↑ Win!
```

### 2 Diagonals:
```
[X][ ][ ][ ][ ]    [ ][ ][ ][ ][X]
[ ][X][ ][ ][ ]    [ ][ ][ ][X][ ]
[ ][ ][F][ ][ ]    [ ][ ][F][ ][ ]
[ ][ ][ ][X][ ]    [ ][X][ ][ ][ ]
[ ][ ][ ][ ][X]    [X][ ][ ][ ][ ]
  ↘ Win!             ↙ Win!
```

## How to Use

1. **Select Pattern:** "Any Line (Horizontal, Vertical, or Diagonal)"
2. **Start Game**
3. **Complete ANY line** (horizontal, vertical, or diagonal)
4. **Call BINGO!**

## Why Use It?

✅ **Fastest games** - 12 ways to win vs 2-5 for other patterns
✅ **Most exciting** - Multiple winning paths
✅ **Beginner friendly** - Easy to understand
✅ **Fair for all** - Everyone has equal chances
