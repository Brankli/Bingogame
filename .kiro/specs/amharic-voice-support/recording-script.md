# Amharic Voice Recording Script
## 75-Ball Bingo Game

---

## Recording Instructions

### For Voice Talent:
- Speak clearly and at a moderate pace
- Maintain consistent volume and tone throughout
- Pause 0.5 seconds between recordings
- Pronounce numbers naturally as they would be said in everyday Amharic
- Use a neutral, friendly tone

### Technical Specifications:
- **Format**: MP3
- **Bitrate**: 128 kbps
- **Sample Rate**: 44.1 kHz
- **Channels**: Mono
- **Duration**: 1-3 seconds per file
- **Silence**: Trim leading/trailing silence to < 0.1 seconds

## Recording Checklist

### Pre-Recording:
- [ ] Test microphone and recording equipment
- [ ] Find quiet recording environment
- [ ] Review pronunciation guide
- [ ] Do vocal warm-up exercises
- [ ] Test recording levels

### During Recording:
- [ ] Record each file separately
- [ ] Maintain consistent distance from microphone
- [ ] Speak clearly and naturally
- [ ] Pause between recordings
- [ ] Mark any files that need re-recording

### Post-Recording:
- [ ] Review all 98 audio files
- [ ] Normalize volume levels
- [ ] Trim silence from beginning and end
- [ ] Convert to MP3 format (128 kbps, 44.1 kHz, mono)
- [ ] Rename files according to naming convention
- [ ] Organize into correct folder structure
- [ ] Test playback quality
- [ ] Create backup of all files
