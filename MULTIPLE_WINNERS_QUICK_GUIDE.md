# ⚡ Multiple Winners - Quick Guide

## The Feature

**Multiple cards can win in the same game!**

## How It Works

### Scenario:
```
Pattern: Horizontal Line
Numbers Called: B-1, B-2, B-3, B-4, B-5, ...

Card #5:  Completes horizontal line ✅
Card #10: Completes horizontal line ✅
Card #15: Completes horizontal line ✅

All three cards win!
```

## Verification Process

### Winner #1:
```
Verify Card #5
✅ "Card CARD-5 wins with Horizontal Line"
Prize: Full amount (for now)
```

### Winner #2:
```
Verify Card #10
✅ "Card CARD-10 wins with Horizontal Line (Winner #2)"
ℹ️ "Total winners: 2 - Prize will be split equally"
Prize per winner: Total ÷ 2
```

### Winner #3:
```
Verify Card #15
✅ "Card CARD-15 wins with Horizontal Line (Winner #3)"
ℹ️ "Total winners: 3 - Prize will be split equally"
Prize per winner: Total ÷ 3
```

## Prize Split

### Formula:
```
Prize per Winner = Total Prize ÷ Number of Winners
```

### Examples:

**1 Winner:**
```
Total: 160 Birr
Winners: 1
Each gets: 160 Birr
```

**2 Winners:**
```
Total: 160 Birr
Winners: 2
Each gets: 80 Birr
```

**3 Winners:**
```
Total: 160 Birr
Winners: 3
Each gets: 53 Birr
```
